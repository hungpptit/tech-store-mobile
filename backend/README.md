# Tech Store Stripe Backend

## Run
```powershell
npm install
npm start
```

## Env
```env
PORT=3000
STRIPE_SECRET_KEY=sk_test_...
```

## Android config
```properties
STRIPE_PUBLISHABLE_KEY=pk_test_...
STRIPE_BACKEND_BASE_URL=https://<your-ngrok-url>
```

## ngrok
```powershell
ngrok http 3000
```
